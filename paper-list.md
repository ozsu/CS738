# Paper Review Reading List

The following are the approved papers for the paper review assignment.

Students should select two papers from different topic areas.

## Instructions to Students

- Select **two papers from different sections**.
- At least **one paper must be a systems paper**.
- Reviews should address:
  - problem formulation,
  - system architecture,
  - core techniques,
  - evaluation methodology,
  - relevance to data engineering for data science.
